# Full-Stack Blog Projesi - Frontend (Next.js & Cypress)

Bu proje, BDD (Davranış Odaklı Geliştirme) ve TDD (Test Odaklı Geliştirme) metodolojileri kullanılarak geliştirilmiş bir blog uygulamasının frontend kısmıdır. Test senaryoları Cypress ve Gherkin dili ile yazılmıştır.

## Teknolojiler
- Next.js 16
- React
- Cypress
- Gherkin (Cucumber)

## Kurulum ve Çalıştırma
1. `git clone https://github.com/aliordekl/blog-frontend.git`
2. `cd blog-frontend`
3. `npm install`
4. `npm run dev` (Uygulamayı çalıştırmak için)
5. `npx cypress open` (Testleri etkileşimli modda çalıştırmak için)
6. `npx cypress run` (Videolu testi çalıştırmak için)

## Değerlendirme İçin
Bu proje, **Nurettin Şenyer** ve **Ömer Durmuş** tarafından verilen ödev kapsamında geliştirilmiştir.
